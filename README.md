# ⚡📊 Flash-Dash

O **Flash-Dash** é o projeto de conclusão de curso desenvolvido para democratizar o acesso a dados. O sistema automatiza processos de Business Intelligence (BI), oferecendo uma solução de dashboard ágil e de baixo custo, focada em Pequenas e Médias Empresas (PMEs) e Analistas de Dados.

## 🛠️ Stack Tecnológica

* **Front-End:** Flutter (UI/UX)
* **Back-End (Motor de BI):** Python
* **Banco de Dados:** Supabase

## 📂 Estrutura do Repositório (Monorepo)

O projeto está estruturado para abrigar tanto o motor de dados quanto a interface no mesmo repositório:

* `/backend` - Motor de BI em Python.
* `/frontend` - Interface de usuário em Flutter.
* `/docs` - Documentação e requisitos do projeto.
